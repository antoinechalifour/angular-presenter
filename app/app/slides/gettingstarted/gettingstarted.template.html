Getting started

app.controller('AppCtrl', ['SlideManager', function( SlideManager){
    SlideManager.slides([
      {
        name: 'hello',
        templateUrl: 'app/hello/hello.template.html'
      },
      {
        name: 'introducing',
        templateUrl: 'app/introducing/introducing.template.html',
        controller: 'IntroducingCtrl'
      },
      {
        name: 'about',
        templateUrl: 'app/about/about.template.html',
        note: 'This is a note.'
      }
    ]);
  }]);

You just need to init your slides in the SlideManager factory. You have to provide :

  • A template / templateUrl
  • A name (url)
  • [opt] a controller
  • [opt] a note for the presenter
